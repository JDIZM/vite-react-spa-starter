## Getting Started

This project uses Volta.sh to manage npm versions. If you don't have it installed please see the [Volta.sh](https://volta.sh/) website for installation instructions.

```
# install dependencies
pnpm install

# start dev server
npm run dev

# build for production
npm run build
```
